3. ADD API endpoints to manage examples dynamically:

POST /api/examples/{channel_key}
   - Body: {"feature_context": "...", "content": "..."}
   - Adds a new example to the in-memory store for that channel (append to FEW_SHOT_EXAMPLES)
   - Returns the updated examples list

GET /api/examples
   - Returns all examples for all channels

GET /api/examples/{channel_key}
   - Returns examples for a specific channel

DELETE /api/examples/{channel_key}/{index}
   - Removes an example by index
   - Returns updated list

The pre-loaded examples will be available immediately on startup. Additional examples can be added at runtime via POST.
