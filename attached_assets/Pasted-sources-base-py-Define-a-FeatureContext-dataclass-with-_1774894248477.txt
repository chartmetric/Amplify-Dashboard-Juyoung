sources/slack_source.py — Implement SlackSource(SourceAdapter). Constructor takes channel_id string. sources/manual_source.py — Implement ManualSource(SourceAdapter). list_recent_features() returns empty list. get_feature_context() accepts title and description keyword args and returns a FeatureContext with source_type="manual".
config.py — Load ANTHROPIC_API_KEY, ASANA_ACCESS_TOKEN, and SLACK_BOT_TOKEN from environment variables using os.environ.get().
app.py — Flask app with these routes:

GET / — renders templates/index.html
GET /api/health — returns JSON with status "ok" and which API keys are loaded (true/false)
GET /api/sources — returns list of available source types ["asana", "slack", "manual"]
GET /api/sources/asana/features — lists features from Asana
GET /api/sources/asana/features/<feature_id> — gets full context for one Asana feature
GET /api/sources/slack/features — lists features from Slack
GET /api/sources/slack/features/<feature_id> — gets full context for one Slack message
POST /api/sources/manual/feature — accepts JSON {title, description}, returns FeatureContext
GET /api/features/<source_type> — unified endpoint that routes to any registered source's list_recent_features()
GET /api/features/<source_type>/<feature_id> — unified endpoint that routes to any source's get_feature_context()
