chartmetric-api Contract

Live mode turns on automatically the moment both CHARTMETRIC_ADMIN_API_BASE_URL and CHARTMETRIC_ADMIN_API_TOKEN are set in env — no extra flag is required. Until then, this admin runs against a local JSON store. ANNOUNCEMENTS_STUB_MODE is reserved as a kill switch: leave it unset, set it truthy only if you need to pin Amplify back to the local store during an incident.

The full REST contract (auth, schema, endpoints, validation rules, error codes, deployment checklist) lives in docs/chartmetric-announcement-admin-api.md.
